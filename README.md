# 🛠️ Install-CumulativeUpdate.ps1

> PowerShell-скрипт для интерактивного выбора, копирования с сетевого ресурса и установки накопительных обновлений Windows (`.msu`) через DISM.

## ✨ Возможности
- 🖥️ Удобное меню выбора обновления с нумерацией
- 📥 Автоматическое копиров `.msu` с сетевой шары в локальную папку
- 🛠️ Установка через `DISM /Online /Add-Package`
- 🛡️ Валидация путей, обработка ошибок и проверка доступности
- ⏸️ Адаптивная пауза при завершении (совместима с консолью, ISE и VS Code)

## ⚙️ Требования
- **ОС:** Windows 10 / 11 / Windows Server 2016+
- **PowerShell:** 5.1 или 7.x
- **Права:** Обязателен запуск **от имени Администратора** (для DISM и записи в системные пути)
- **Сеть:** Читаемый доступ к указанному UNC-пути с обновлениями

## 📝 Настройка
Перед запуском отредактируйте переменные в начале скрипта:

| Переменная | Описание | Пример |
|------------|----------|--------|
| `$serverPath` | Сетевой путь к папке с `.msu` | `"\\\\fs01\\updates\\2026"` |
| `$Updates` | Массив имён файлов (строгий порядок!) | `@("KB5014032.msu", "KB5023456.msu")` |
| `Write-Host "..."` | Текст меню (должен соответствовать порядку массива) | `"1. Май 2026 (KB5014032)"` |
